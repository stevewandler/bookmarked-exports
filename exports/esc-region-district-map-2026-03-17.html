Methodology: Deals were searched in HubSpot using three query terms: "ESC", "Region", and "Consortium". ESC region numbers were extracted from deal names using pattern matching (ESC Region XX, ESC XX, Region XX). Deal stage IDs were mapped to labels based on pipeline context. Deals with $0 amounts in consortium groups typically indicate bundled pricing through the ESC partnership deal rather than individual district billing.

Stage Labels: Active Partner = ESC partnership pipeline active stage. Inactive/Lost = ESC partnership pipeline closed/inactive. Renewal Pipeline = 25/26 renewal tracking. Closed Won/Lost = Standard sales pipeline outcomes. Negotiation = Active negotiation stage.

Data Gaps: Districts that purchased through direct sales (no ESC reference in deal name) are NOT included here. This report only captures deals with explicit ESC/Region/Consortium references. The actual district count per ESC region may be higher.
