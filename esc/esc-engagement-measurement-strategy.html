ESC Engagement Measurement Strategy

How Bookmarked measures ESC partnership success — from first click to district activation

Internal Strategy Document • March 2026 • Bookmarked

1. What We're Measuring and Why

The ESC partnership channel is Bookmarked's primary distribution strategy. We need to measure three things:

ESC Activation

Are ESC partners actively introducing Bookmarked to their districts? How engaged is each ESC in the partnership?

District Conversion

When ESCs introduce Bookmarked, do districts convert? What's the path from introduction to signed contract?

Channel ROI

What's the cost per district acquired through ESC vs. direct sales? Is the channel scaling?

The core question: How do we get ESCs to truly partner with us — not just sign an agreement, but actively champion Bookmarked to their districts?

2. The ESC Partnership Funnel

Each ESC relationship moves through a predictable funnel. We track conversion rates between stages and time-in-stage to identify bottlenecks.

1. Awareness ESC knows Bookmarked exists. Initial contact made. HubSpot: Lead
2. Agreement ESC signs partnership agreement. Commission structure agreed. HubSpot: Partner
3. Enablement ESC staff trained via Partner Academy. Co-branded assets delivered. Trained
4. Activation ESC makes first district introduction. Sends first email/presents at meeting. Active
5. Revenue First district converts through ESC channel. Commission earned. Revenue

Key Conversion Metrics

Stage Transition Metric Target Data Source
Awareness → Agreement Partnership close rate 60%+ HubSpot deals
Agreement → Enablement Training completion rate 80%+ within 30 days Partner Academy / HubSpot
Enablement → Activation First-introduction rate 70%+ within 60 days HubSpot + ESC landing page analytics
Activation → Revenue ESC-sourced conversion rate 25%+ of introduced districts HubSpot attribution
Time: Agreement → First Revenue Time to first commission < 90 days HubSpot deals + invoicing

3. Landing Page Analytics Architecture

Each ESC landing page is instrumented with comprehensive tracking. Here's what we capture and how it maps to business decisions.

UTM Parameter Strategy

Every link an ESC sends to districts includes UTM parameters so we know exactly which ESC, which campaign, and which asset drove the traffic.

Parameter Value Format Example Purpose
utm_source esc-{region} esc-10 Which ESC partner sent the traffic
utm_medium email | presentation | meeting | direct email How the ESC shared the link
utm_campaign {campaign-name} spring-2026-outreach Which campaign or initiative
utm_content {asset-type} one-pager-risk Which specific asset included the link
utm_term {audience} superintendents Target audience segment

On-Page Event Tracking

All three landing page variants (A, B, C) track identical events so we can A/B test across them.

Event Trigger Properties Captured Business Question Answered
Page View Page load page_variant, utm_*, referrer Which ESCs are driving traffic?
CTA Click Any CTA button click cta_text, cta_url, page_variant, utm_source Which CTAs convert? Which variant performs best?
Scroll Depth 25%, 50%, 75%, 100% depth, page_variant How far do visitors read? Where do they drop off?
Time on Page 30s, 60s, 2min, 5min duration, page_variant Are visitors engaging deeply or bouncing?
Section Visible Section enters viewport section_id, page_variant Which content sections get attention?
Form Start First form field focus page_variant, utm_source Conversion intent signal
Form Submit HubSpot form submission page_variant, utm_*, form_id Actual conversion
Walkthrough Booked Meetings link clicked page_variant, utm_source High-intent conversion

Page Variant Comparison Framework

We're testing three landing page concepts to find the optimal approach for ESC-driven traffic:

Variant Strategic Angle Hypothesis Primary KPI
Option A: Trusted Network Social proof + trust ESC endorsement builds credibility fastest CTA click rate
Option B: Library Vitality Mission + emotion Librarians respond to empowerment narrative Scroll depth + time on page
Option C: Intelligence Advantage Product + data differentiation Decision-makers need the "why different" case Walkthrough bookings

A/B Testing Plan: Initially, send all ESC traffic to a single page (likely Option A for broad appeal). Once we have 200+ visits, split traffic across variants using HubSpot's adaptive page testing. We need ~100 visits per variant for statistical significance on CTA click rate.

Workflows to Build

  1. ESC Source Attribution: When contact visits ESC landing page with utm_source=esc-{N}, set esc_source_region and esc_first_touch_date automatically.
  2. Engagement Scoring: Increment esc_engagement_score based on events: page view (+1), scroll 50%+ (+2), CTA click (+5), form submit (+10), walkthrough booked (+20).
  3. ESC Activity Alerts: When an ESC's districts show landing page activity, notify Jan (jan@bookmarked.com) so she can follow up.
  4. Weekly ESC Partner Report: Auto-send each ESC partner a weekly summary of their districts' engagement (visits, form fills, bookings).
  5. Stale Partner Alert: If an ESC partner hasn't driven any traffic in 30 days, alert Jan to re-engage.

5. ESC Partner Scorecard

Each ESC partner gets a monthly scorecard measuring their engagement and effectiveness. This is shared with them to reinforce partnership value and motivate activity.

Scorecard Metrics

Metric What It Measures Data Source Healthy Range
Districts Reached Unique districts that visited the landing page via this ESC's UTM links HubSpot + GA 5+ / month
Walkthroughs Booked Number of districts that booked a walkthrough from ESC-sourced traffic HubSpot meetings 2+ / month
Districts Converted Number of closed deals attributed to this ESC HubSpot deals 1+ / quarter
Pipeline Value Total deal value in pipeline from ESC-sourced districts HubSpot deals Growing quarter over quarter
Commission Earned Total commission paid to this ESC partner Invoicing / HubSpot Grows with conversions
Asset Usage Which co-branded assets are being shared (tracked via UTM content parameter) UTM tracking 2+ asset types used
Partner Academy Completion % of ESC staff who completed training modules Partner Academy LMS 80%+ completion

ESC Health Score

Each ESC gets a composite health score (0-100) based on weighted scorecard metrics:

Activation (40%)

Are they sending traffic? Using assets? Making introductions?

Conversion (35%)

Are their districts converting? Booking walkthroughs? Signing contracts?

Engagement (25%)

Training completion, response time, asset feedback, meeting attendance.

7. Implementation Roadmap

Phase 1: Foundation (This Week)

  • Landing pages live with tracking — All three variants deployed to HubSpot CMS with full analytics instrumentation
  • UTM links generated — Create pre-built UTM links for each ESC partner and each asset type
  • HubSpot custom properties created — esc_source_region, esc_landing_page_variant, esc_engagement_score

Phase 2: Attribution (Week 2)

  • Source attribution workflow — Automatically tag contacts by ESC source
  • Engagement scoring workflow — Score contacts based on landing page behavior
  • Deal attribution — Link ESC-sourced contacts to deals for commission tracking

Phase 3: Reporting (Week 3-4)

  • HubSpot dashboard — Real-time ESC channel dashboard with all key metrics
  • Weekly report automation — Auto-generated weekly email to Steve + Jan
  • Partner scorecard template — Monthly scorecard for each ESC partner

Phase 4: Optimization (Ongoing)

  • A/B test landing page variants — Determine winning page and roll out
  • Refine engagement scoring — Adjust weights based on actual conversion data
  • Partner re-engagement automation — Proactive outreach for stale partners
  • Regional performance analysis — Compare ESC effectiveness by region

Success looks like: Within 90 days, we can answer "Which ESC partner is most effective?", "Which landing page converts best?", "What's our ESC channel CAC?", and "Which districts are closest to converting?" — all from a single HubSpot dashboard.
